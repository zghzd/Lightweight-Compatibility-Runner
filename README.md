# Lightweight Compatibility Runner (LCR)
**不保证**应用程序行为的正确性，
也不建议用于完整运行应用程序。  
但如果你正在进行**动态逆向分析**，
LCR不失为一个有用的工具。

*你可能需要预处理一部分文件，之后才可以使用LCR.详情请见每个功能的README，而不是此README*
